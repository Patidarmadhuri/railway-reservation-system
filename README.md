# Online Railway Reservation System

A simple full-stack railway reservation system built with Java Spring Boot backend and a lightweight HTML/CSS/JavaScript frontend. Users can search trains, book tickets, and view booking history.

## Features
- Search trains by origin, destination and date
- Book tickets with passenger details and seat count
- View booking history for a user
- REST API endpoints under /api for trains and reservations
- Separate backend (Spring Boot) and frontend (static HTML/CSS/JS)

## Tech Stack
- Backend: Java, Spring Boot (web, JPA), Hibernate, H2 (or MySQL/PostgreSQL)
- Frontend: HTML, CSS, JavaScript
- Build: Maven

## Running Locally
Clone the repo and run the backend via `./mvnw spring-boot:run` in the `backend` directory (localhost:8080/api). Open `frontend/index.html` in your browser and adjust API base URL in `script.js` if needed.

## Structure
`backend/` contains Java sources and `frontend/` holds the static site. See files for details.
